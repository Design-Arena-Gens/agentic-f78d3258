# Wildberries-style Product Card

Next.js demo of a Wildberries-like product card based on the linked Uzum product for content inspiration. Includes image gallery, color/size selection, pricing, delivery info, and structured data.

- Build: `npm install && npm run build`
- Start: `npm start`
- Deploy (Vercel): `vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-f78d3258`
